This folder "src\components\general" holds all general visual components that are used throughout
the user interface of this web application

These components are built as react functional components with any state being managed through
the use of react hooks.

CSS Styling for each component is supplied via css module files that mirror the file names of
the corresponding javascript component file. 

Components
----------------------------------------------------------------------------------------------
BoxSelector.js

This component is input component used denote a selected item. (Switch Component).

----------------------------------------------------------------------------------------------
Button.js

This is a visual input component Button.

----------------------------------------------------------------------------------------------
Footer.js

This component makes the footer component of this web application.

----------------------------------------------------------------------------------------------
Navbar.js

This component makes the Navbar component of this web application.
-----------------------------------------------------------------------------------------------
Navbar Login.js

This component makes the Navbar component to be used when users are logged into this web application.

-------------------------------------------------------------------------------------------------
SearchInputField.js

This is an visual input component used to take in text input.
